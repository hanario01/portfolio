# Urban Pause

## 概要
「Urban Pause」は、  
**都会の喧騒を忘れ、静かに過ごせるカフェ**をコンセプトに制作した架空のカフェサイトです。

UIデザイン・アニメーション表現・構造設計を中心に、  
**実務を意識したコーディング**を意識して制作しています。

---

## コンセプト
**都会にいながら、一息つける場所。**

- 落ち着いた配色
- 余白を活かしたレイアウト
- 控えめなアニメーション

を通して、  
「視覚的な静けさ」「心地よいテンポ」を感じられるサイトを目指しました。

---

## 使用技術
- HTML5  
  - セマンティックな構造を意識したマークアップ
- CSS3  
  - CSS変数による配色管理  
  - レスポンシブ対応（SP / PC）
  - アニメーション・トランジション表現
- JavaScript（Vanilla JS）  
  - スクロール連動アニメーション  
  - クラス付け替えによるUI制御

※ フレームワークは使用せず、基礎技術での実装を行っています。

---

## 実装内容
- ファーストビューのタイピング風アニメーション
- スクロールに応じたフェードイン / スライドイン演出
- Menuカードの hover スケール表現
- SP / PC で表示を切り替えるUI設計
- Google Map 埋め込み
- SNSリンク設置

---

## コーディングで意識した点
- **CSSとJavaScriptの責務分離**
  - `.js-` プレフィックスを用いたJS専用クラス設計
- hover用・アニメーション用クラスを分離し、競合を防止
- デザイン変更に耐えやすい構造を意識
- 実務を想定し、後から修正しやすいクラス設計

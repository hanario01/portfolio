# .Anew Hair Salon（美容室サイト）

## 概要
実在する美容室サイトを想定し、  
**ブランドイメージ・導線設計・UI表現** を重視して制作したWebサイト(架空)です。

「日常の小さな変化が、明日への自信につながる」というコンセプトを軸に、  
落ち着いた配色・余白設計・タイポグラフィを意識しています。

---

## デザインURL
https://www.figma.com/design/Z9ufVSPeS7AutfPWEH8I53/.Anew?node-id=0-1&t=1Ompmj9CYnffGnx9-1

## 制作目的
- 実務を想定した **中規模サイト構成の理解**
- HTML / CSS / JavaScript を組み合わせたUI実装練習
- デザインカンプを意識したコーディング精度の向上
- コンテンツ量の多いサイトでの **クラス設計・再利用性** の検証

---

## 使用技術
- HTML5（セマンティック構造を意識）
- CSS3  
  - CSS変数による色管理  
  - clamp() を使用したレスポンシブタイポグラフィ  
  - 疑似要素・clip-path を使った装飾表現
- JavaScript（Vanilla JS）
  - ハンバーガーメニュー制御
  - スライダー切り替え
  - タブ切り替えUI

---

## 実装している主な機能・UI
- ファーストビューのビジュアル設計
- 固定ナビゲーション＋ハンバーガーメニュー
- REASON セクションのスライド切り替え
- SERVICE セクションのタブUI
- ヘアスタイルの横スクロールアニメーション
- 視線誘導を意識した背景テキスト装飾

---

## 設計・実装で意識した点
- 実案件を想定し、**セクション単位での構造分離**
- クラス名は「見た目」ではなく「役割」ベースで命名
- 拡張や修正が入りやすいことを前提にしたHTML構造
- デザイン要素が強い部分でも、保守性を意識したCSS設計
